Habilidades

HTML

Meu primeiro contato com o mundo do desenvolvimento web se deu através do HTML, uma linguagem de marcação essencial que serve como a base estrutural de qualquer página na internet. A partir do momento em que comecei a entender como funcionava a criação de páginas estáticas, pude visualizar o imenso potencial da tecnologia na construção de sites e aplicações. O HTML me proporcionou um entendimento profundo sobre a organização da web, permitindo que eu compreendesse como os navegadores processam e exibem informações. Mais do que uma simples linguagem de marcação, o HTML foi um catalisador para despertar minha curiosidade sobre o desenvolvimento e, consequentemente, sobre como otimizar a experiência dos usuários na navegação online. Através dele, aprendi que a estrutura e semântica são fundamentais para a acessibilidade e SEO, dois fatores cruciais no desenvolvimento moderno. Esse aprendizado me impulsionou a seguir uma carreira na tecnologia, com foco no desenvolvimento de soluções que não apenas funcionam, mas também criam valor para os usuários e empresas.

CSS

Se o HTML me deu a estrutura, o CSS foi o elemento que me permitiu dar vida às páginas. Essa linguagem de estilização foi o primeiro passo para que eu pudesse expressar minha criatividade no desenvolvimento web. Desde o início, percebi que o CSS era muito mais do que apenas cores e fontes; ele se tratava de criar experiências visuais envolventes, otimizadas e acessíveis. Através dele, pude aprender sobre a importância de design responsivo, flexibilidade e usabilidade, além de compreender como um bom layout pode afetar diretamente a forma como os usuários interagem com uma página. Com o tempo, fui aperfeiçoando minhas habilidades, utilizando ferramentas como media queries e flexbox para garantir que meus projetos fossem adequados a diferentes dispositivos e tamanhos de tela. A combinação de HTML e CSS me fez entender que a estética e a funcionalidade devem andar lado a lado, e essa compreensão me levou a explorar ainda mais a fundo o design de interfaces de usuário (UI) e a experiência do usuário (UX). A estilização das páginas, portanto, tornou-se uma maneira de expressar minha paixão por arte, e cada projeto se transformou em uma extensão da minha visão criativa.

JavaScript

Ao dominar HTML e CSS, percebi que a próxima etapa para evoluir no desenvolvimento web seria mergulhar no mundo da programação, e JavaScript foi a linguagem que me abriu as portas para isso. A transição de linguagens de marcação para uma linguagem de programação de verdade foi desafiadora, mas ao mesmo tempo fascinante. Inicialmente, me deparei com uma curva de aprendizado íngreme, onde precisei aprender a lógica de programação, manipulação de DOM e eventos, além de conceitos mais avançados como promessas e assincronismo. No entanto, essa dificuldade inicial logo se transformou em entusiasmo, pois vi o impacto imediato que JavaScript podia causar em uma página. A linguagem me permitiu adicionar interatividade e dinamismo aos meus projetos, fazendo com que as páginas não fossem apenas estáticas, mas interativas e envolventes. Além disso, JavaScript também abriu caminho para novas oportunidades de aprendizado, como a exploração de frameworks populares como React e a integração com APIs. Foi a partir de JavaScript que comecei a desenvolver um interesse mais profundo por back-end, performance e otimização de aplicações web, temas que hoje considero essenciais para o desenvolvimento de soluções completas e robustas.

Csharp

Após adquirir uma base sólida em JavaScript, minha trajetória no desenvolvimento de software deu um grande salto quando tive a oportunidade de trabalhar com C#. Esta linguagem de programação orientada a objetos, amplamente utilizada em diversos tipos de aplicações, tornou-se um divisor de águas em minha carreira. Durante minha experiência em uma empresa, desenvolvi aplicações desktop utilizando o framework .NET 6.0, o que me permitiu aplicar conceitos avançados de programação, como encapsulamento, herança e polimorfismo, além de padrões de design que aumentaram a qualidade do código. Foi nesse ambiente corporativo que pude ver, na prática, o impacto que uma linguagem robusta como o C# pode ter no desenvolvimento de soluções empresariais de alta performance. O aprendizado foi intenso, mas também recompensador, pois o C# me ensinou não apenas a codificar, mas a pensar de forma mais estratégica, considerando aspectos como escalabilidade e manutenibilidade das aplicações. Hoje, vejo o C# e a plataforma .NET como peças-chave no meu desenvolvimento como programador. Meu objetivo futuro é alcançar o reconhecimento como MVP pela Microsoft, um título que reforçaria meu compromisso com a excelência e inovação dentro do ecossistema .NET.

MySql

Quando comecei a trabalhar com C#, rapidamente percebi a importância de dominar o gerenciamento de bancos de dados, e foi assim que tive meu primeiro contato com o MySQL. Utilizando o MySQL Workbench como ferramenta, fui capaz de aprender as nuances de uma das linguagens mais poderosas para manipulação de dados: SQL. O gerenciamento de banco de dados é essencial em qualquer aplicação moderna, e essa habilidade me permitiu criar sistemas robustos e eficientes. Em especial, apliquei meus conhecimentos em um sistema CRUD, onde desenvolvi uma solução completa para a criação, leitura, atualização e exclusão de dados, o que me permitiu integrar o front-end com o back-end de maneira eficaz. O trabalho com bancos de dados não só ampliou minhas capacidades técnicas, mas também me proporcionou uma compreensão mais profunda da arquitetura de sistemas, otimização de consultas e segurança de dados, elementos críticos para o sucesso de qualquer aplicação de larga escala.

Asp.Net Core

Atualmente, tenho me dedicado com afinco ao estudo e ao aprimoramento das minhas habilidades em ASP.NET Core, um framework da plataforma .NET que me proporciona uma maneira eficiente de desenvolver aplicações web escaláveis e de alta performance. Minha experiência com este framework engloba a utilização de tecnologias como MVC, Razor Pages e Blazor, permitindo que eu construa soluções completas para o ambiente web. O que mais me atrai no ASP.NET Core é a sua flexibilidade e poder, especialmente ao trabalhar com o .NET 8.0. No entanto, ao explorar o conteúdo disponível sobre essa tecnologia, percebi que ainda há uma lacuna significativa de materiais voltados para iniciantes, e é exatamente essa necessidade que me motiva a, no futuro, contribuir com a comunidade. Pretendo compartilhar meu conhecimento e ajudar a preencher essa lacuna, criando conteúdos didáticos que possam auxiliar outros desenvolvedores que estão iniciando sua jornada no ASP.NET Core, facilitando sua aprendizagem e capacitação para o mercado de trabalho.

Git

Git foi uma das primeiras ferramentas que introduzi na minha rotina de desenvolvimento e rapidamente percebi sua importância para o versionamento de código. Com ele, aprendi a gerenciar diferentes versões de um projeto, permitindo que eu trabalhasse de forma mais organizada e eficiente. Recentemente, obtive uma certificação em Git pela plataforma DIO.me, o que consolidou ainda mais meu entendimento sobre o funcionamento dessa poderosa ferramenta. Git não é apenas sobre controle de versão; ele também permite colaboração eficiente em equipes, já que diferentes desenvolvedores podem trabalhar simultaneamente no mesmo projeto, realizando commits, gerenciando branches e resolvendo conflitos de maneira rápida. A experiência que adquiri utilizando Git foi fundamental para que eu pudesse voltar no tempo e recuperar versões anteriores de códigos importantes que haviam sido alterados acidentalmente. A ferramenta provou ser indispensável para garantir a integridade e a continuidade de qualquer projeto de software.

Bootstrap

O primeiro framework CSS que tive o prazer de aprender. Comecei a utilizá-lo durante um projeto importante em que trabalhei. Com a versatilidade do Bootstrap, pude aplicar componentes prontos que aceleraram o processo de desenvolvimento front-end. A modularidade e a responsividade nativas foram recursos essenciais, facilitando o desenvolvimento de layouts responsivos com menor esforço. No entanto, para otimizar a performance de um projeto específico, decidi seguir por uma abordagem manual, utilizando HTML, CSS e JavaScript puros, sem o suporte do framework. Apesar de exigir mais trabalho, essa decisão me permitiu entender melhor a estrutura do código e aumentar minhas habilidades com estilização personalizada. O aprendizado com Bootstrap, no entanto, segue sendo fundamental para mim, especialmente quando busco agilidade em protótipos ou projetos que necessitem de rápida implementação de design moderno e responsivo.

React

React é um dos frameworks front-end mais poderosos com os quais comecei a ter contato. A flexibilidade e a estrutura baseada em componentes do React oferecem uma maneira eficiente de construir interfaces dinâmicas e interativas. Ainda que eu não tenha desenvolvido um projeto significativo com ele até o momento, estou profundamente interessado em explorar mais suas capacidades. Pretendo integrar o React em uma aplicação web utilizando C# no back-end, combinando o melhor de ambos os mundos. Além disso, vejo no React uma excelente oportunidade para aprimorar a experiência do usuário com interatividade fluida e rápida, aproveitando o Virtual DOM para otimizar a performance. A comunidade vibrante em torno do React também é uma grande fonte de suporte, tornando o aprendizado contínuo desse framework uma jornada empolgante e promissora para o meu desenvolvimento profissional.
